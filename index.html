Dual-Layer Risk Decision Correlation flow

Correlated Decision Rules:

• If a process attempts to access a deployed **Honeytoken Decoy** (Layer 2), the system assigns it a threat score of **45%** and triggers a silent administrator warning.

• If a process exhibits **abnormal write Shannon entropy** above the threshold (Layer 1), the system immediately marks it as **Critical** and issues a termination call.

• Together, the two layers guarantee that zero-day ransomware cannot execute mass file encryption undetected, minimizing both false positives and detection latency.
